│                    TECHNOLOGIES UTILISÉES                   │
└─────────────────────────────────────────────────────────────┘

Backend:
  • Python 3.9+
  • FastAPI (API REST)
  • Motor (MongoDB async)
  • Pydantic (validation)
  • JWT (authentification)
  • ReportLab (PDF)
  • OpenPyXL (Excel)

Frontend:
  • React 18
  • JavaScript ES6+
  • Tailwind CSS
  • Axios (HTTP)
  • FontAwesome (icônes)

Base de données:
  • MongoDB
  • Collections: users, plannings, formations, etc.

DevOps:
  • Docker & Docker Compose
  • Git & GitHub
  • Vercel (hosting frontend)
  • Render (hosting backend)
  • MongoDB Atlas (DBaaS)


┌─────────────────────────────────────────────────────────────┐
│                    COMMANDES RAPIDES                        │
└─────────────────────────────────────────────────────────────┘

🐳 Docker (Recommandé):
   docker-compose up -d              # Démarrer tout
   docker-compose logs -f            # Voir les logs
   docker-compose down               # Arrêter tout

🐍 Backend local:
   cd backend
   source venv/bin/activate
   uvicorn server:app --reload

⚛️ Frontend local:
   cd frontend
   yarn start

👤 Créer admin:
   python backend/create_admin.py

🧪 Tests:
   pytest backend/                   # Backend
   cd frontend && yarn test          # Frontend
